# 📋 TaskFlow – Task Management Web App (Task 4)

## 📌 Description

TaskFlow is a browser-based task management web application built using **HTML, CSS, and JavaScript** as part of the SkillCraft Technology Web Development Internship.

It helps users organize and manage daily tasks efficiently.

---

## 🚀 Features

* Add, edit, and delete tasks
* Task status management (pending, in progress, completed)
* Priority handling
* Clean and user-friendly interface
* Local storage support

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript

---

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
```

---

## ▶️ How to Run

1. Clone or download the repository
2. Open `index.html` in your browser
